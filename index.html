Rules



The object of the game is to identify sets of 3 cards from 12 cards placed on the board. The game ends when all of the cards have been dealt and there aren't any more sets.

Each card has 4 attributes:

Number
  • One
  • Two
  • Three
Color
  • Red
  • Green
  • Purple
Shape
  • Circle
  • Triangle
  • Square
Fill
  • Solid
  • Striped
  • None

A set consists of 3 cards where each of the cards' attributes are either the same or different. In other words, the three cards must either all have the same shape or all have a different shape, and the three cards must either all be the same color or all be different colors. If two cards have the same attribute and one card has a different attribute, it is not a set.
